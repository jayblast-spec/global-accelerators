## ✨ Features

- 🌐 **164+ verified programs** — YC, Techstars, 500 Startups, regional cohorts, and niche verticals
- 🤖 **AI application templates** — tailored prompts and drafts matched to each accelerator's criteria
- 🔍 **Filter by stage, sector, geography** — find the programs that fit your exact startup
- 📅 **Deadline tracking** — active intake windows surfaced first
- 📝 **One-click template generation** — AI writes your first draft based on your startup description
- 🏆 **Program reputation scores** — alumni outcomes, check sizes, and equity terms in one place
- ⚡ **No signup required** — search and generate immediately

## 🚀 Quick Start

```bash
git clone https://github.com/jayblast-spec/global-accelerators.git
cd global-accelerators
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — or use the [live version](https://global-accelerators.vercel.app).
